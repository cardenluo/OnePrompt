# 积木提示词

一个Chrome浏览器扩展,提供通用提示词管理功能,支持快捷键弹窗、右键收藏、批量导入导出CSV等。**可在所有网页使用!**

## 功能特性

- ⌨️ **快捷键支持**: 使用 `Alt+Q` 快速打开/关闭提示词侧边栏
- 🌐 **全网页支持**: 在任何网页都可以使用(不限于豆包)
- 🖱️ **右键收藏**: 选中文本后右键"收藏为提示词"
- 📝 **提示词管理**: 增删改查提示词,支持分类和标签
- 🔍 **快速搜索**: 实时搜索提示词标题、内容、分类和标签
- 🏷️ **分类筛选**: 自动生成分类下拉框,快速筛选
- 📥 **批量导入**: 支持CSV格式批量导入提示词
- 📤 **批量导出**: 一键导出所有提示词为CSV文件
- 📌 **固定侧边栏**: 可固定侧边栏,同时操作网页
- 🎨 **现代UI**: 美观的侧边栏界面,流畅的动画效果
- 🚀 **一键使用**: 点击使用按钮直接复制并插入到输入框

## 安装方法

1. 下载或克隆此项目到本地
2. 打开Chrome浏览器,访问 `chrome://extensions/`
3. 打开右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择本项目的文件夹

## 使用方法

### 基本操作

1. 在**任何网页**按 `Alt+Q` 打开提示词侧边栏
2. 点击"新建提示词"创建提示词
3. **右键收藏**: 选中文本 → 右键 → "收藏为提示词"
4. 点击提示词卡片上的"使用"按钮复制并插入到输入框
5. 点击📌按钮可固定侧边栏

### CSV导入导出

#### CSV文件格式

CSV文件应包含以下列(第一行为标题):

```
标题,内容,分类,标签
```

示例:
```csv
标题,内容,分类,标签
代码优化专家,"请帮我优化以下代码,提高性能和可读性:",编程,代码|优化
文案创作,"请帮我创作一段关于以下主题的文案:",写作,文案|创作
```

**注意**:
- 标签使用竖线 `|` 分隔
- 如果内容包含逗号或换行,需用双引号包裹
- 使用UTF-8编码保存CSV文件

#### 导入步骤

1. 打开提示词弹窗
2. 点击"导入CSV"按钮
3. 选择CSV文件
4. 导入成功后会显示导入数量

#### 导出步骤

1. 打开提示词弹窗
2. 点击"导出CSV"按钮
3. 文件会自动下载,文件名格式: `one_prompts_YYYY-MM-DD.csv`
